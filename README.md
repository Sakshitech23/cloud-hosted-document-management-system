## Project Description
This project is a cloud-hosted document management system designed to support secure file upload, centralized storage, secure sharing, file version tracking, access management, and high availability.

## Main Features
- Secure file upload and retrieval
- Centralized cloud storage
- Access permission control
- File version tracking
- Storage monitoring
- Large file handling
- Security-focused architecture
